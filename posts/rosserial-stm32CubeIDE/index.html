Part - 2 Stm32CubeIDE

Send this Inc folder with all the repositories to yourself through email/ usb. Rename this folder as ‘ROS’ so that the Inc folders in stm32CubeIDE does not get confused with this newly generated Inc folder using ROS.

Place this folder inside the stm32CubeIDE project : ` “Your Project name” -> Core -> Inc -> ROS` This ROS folder is copied inside the stm32CubeIDE. If you don’t from chatter project Inc. Copying the ROS in the files folder of Windows directory does not work.

After the header files and source code file mainpp.h and mainpp.cpp are added along with necessary changes in main.c and hardware.h then Debug the program as ros_test1.elf.

Connect the stm32f303re board to the laptop. Right click the project and click Debug as -> configurations -> debug (on the dialog box)
